This is a homage to the first website I remember building.

It was back in the 90's, probably on Geocities and I was trying to make a virtual space station where each section of the space station took you to a different page on the site.

Google wasn't around and the only way I could learn Javascript was from snippets on a CD-ROM from one of my Dad's PC mags.

I never quite managed to get the image map or Javascript working well enough to make the site work.

In memory to those days I decided to build this site similarly to how I tried to build sites back then:

  • One file
  • No libraries
  • No build tools

I let myself use Google this time though...
